Word模板填充打印软件

运行要求：
	1、需要Window7及以上的操作系统
	2、需要安装Microsoft Office2007及以上版本
	3、需要.NetFramework 4.5.2及以上版本

说明：
	1、本软件只是自用辅助小工具，不得商用，出现任何问题概不负责
	2、使用Office的Word作为模板，在模板中使用书签作为填充位，每个书签的名称不能相同
	3、只支持文本的替换，不支持图片替换
	4、替换的文本只能是字符串。暂时不支持数字等其它格式，但可以先将其它格式手动转换成字符串
	5、模板文件需要精心编辑，书签的位置至关重要，前后空格、标点符号等都会影响最终质量

操作步骤：
	1、双击桌面上的快捷方式，打开软件
	2、点击界面上的【打开】按钮，选择Word模板文件
	3、等待文件读取完成。文件读取后，列表中会列出所有的书签名称
	4、如果模板有问题，可打开模板进行编辑，编辑完成后，一定要保存模板，并在软件界面上点击【刷新】按钮
	5、在【添加】按钮后面有个输入框，在这个输入框里输入需要的列表行数，必须为正整数。列表的每一行数据配合模板就可生成一个新的文件
	6、点击【添加】按钮，列表中会多出相应的行
	7、在列表中编辑每行的数据，每列对应着模板中的每个书签
	8、【清除所有】按钮表示清空列表中的所有数据，从头再来
	9、【清除空行】按钮表示清空列表中的空白行。空白行如果不清除，导出和打印时依然会使用该行数据，只是导出和打印出来的文件没有填充数据
	10、列表中的【删除】按钮表示删除该行数据
	11、【导出数据】可将列表中数据导出成一个Excel文件
	12、【导入数据】可将上一步导出的Excel文件里的数据导回到列表中，可重复多次导入，列表数据也会重复增加
	13、导出文件名可选择列表的列名，在【导出】功能中，会使用该列数据作为导出文件的文件名。如果不选中，就使用数字进行命名，从0开始
	14、【导出】按钮是使用列表中的数据，填充模板后，保存到电脑上的某个目录下
	15、【打印】前可设置单双面打印和打印份数这两个参数。双面打印需要打印机的支持
	16、【打印】按钮是使用列表中的数据，填充模板后，打印到选择的打印机上
